# Markdown Templates

Reusable Markdown templates for AMA scientific document authoring.

- `decision_record.md`: concise decision and rationale capture.
- `how_to_guide.md`: procedural scientific or operational instructions.
- `issue.md`: problem report structure.
- `kanban.md`: lightweight work tracking.
- `presentation.md`: text-first slide outline.
- `project_documentation.md`: project reference documentation.
- `pull_request.md`: change review summary.
- `research_paper.md`: research manuscript scaffold.
- `status_report.md`: progress and risk report.
