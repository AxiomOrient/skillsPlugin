# Scientific Document Skills

This directory contains AMA mobile document skills grouped by runtime behavior.

- `markitdown`: routes supported text-like artifacts through AMA Swift document-to-Markdown intents.
- `markdown-mermaid-writing`: provides Markdown and Mermaid writing guidance plus reusable templates.
- `docx`, `pptx`, `xlsx`: provide mobile-safe planning guidance for Office formats until native Swift editors exist.

PDF-specific handling is intentionally not installed in this mobile plugin. Convert binary documents to Markdown or another supported UTF-8 text-like format before importing them into AMA.
