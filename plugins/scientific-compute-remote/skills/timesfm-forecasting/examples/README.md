# TimesFM Examples

The examples in this directory are reference artifacts for planning remote TimesFM jobs on AMA iOS.

- `global-temperature`: forecast report, input CSV, and generated output artifacts.
- `anomaly-detection`: generated anomaly output artifacts.
- `covariates-forecasting`: generated covariate example artifacts.

AMA iOS does not run TimesFM, Python, shell scripts, package managers, notebooks, or model downloads locally. Use `scientific.remote_job_preflight` before claiming execution.
