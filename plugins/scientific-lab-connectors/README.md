# Scientific Lab Connectors

AMA mobile plugin category for lab SaaS, ELN, microscopy, genomics cloud, cloud-lab, and lab automation connector guidance.

Version `0.1.0` exposes one AMA Swift host intent:

- `scientific.lab_connector_preflight`

The intent evaluates credentials, mutating behavior, user approval, and hardware/remote-execution requirements before a skill suggests any external connector action.

This pack does not ship Python, shell, Node, PowerShell, batch, Docker, local SDK, robot-control, or desktop helper scripts.
